What is Base?

Base is a system built to help Ethereum work faster and cheaper. It handles some of the tasks away from the main Ethereum network, making it less crowded and less expensive to use, while still staying safe and connected to Ethereum.

Analogy of Base

Think of Ethereum as a busy highway where cars represent individual transactions. Each car can only carry a few passengers, so when too many cars try to travel at once, the road gets congested. Now, imagine buses that can carry many passengers at once. When there are a lot of transactions to process, these buses can take more transactions and send them to Ethereum efficiently. This helps clear the road and keeps traffic flowing smoothly for everyone.

Coinbase Wallet

Coinbase Wallet is like the Gcash of Web3. This is where we store and hold crypto assets like Bitcoin, Ethereum, and Slime. It also allows you to browse and connect to Web3 applications, send and receive other cryptocurrency tokens, and buy or collect NFTs.

We use Ethereum for every transaction we make, similar to processing fees on banking apps.

Note:Always make sure to use the browser from Coinbase Wallet when interacting with Web3 Application

Base Names

Every Coinbase Wallet has its own unique ID called a wallet address. This address is made up of random characters and numbers, making it hard to remember. That’s where Base Name comes in. Instead of using the long wallet address, you can use a simpler username (like jesse.base.eth) as your identity on the blockchain. This makes it much easier to send crypto tokens to your friends!
